# LLD-Commerce
Low Level System Design Concepts covered over E-Commerce App
State: The "Baton" logic.
Strategy: The "Payment Tool."
Observer: The "Notification Shout."
Factory: The "Team Manager" (Setup) that works with State.
Facade: The "Front Desk" (Simplicity).
Singleton: The "Global Settings" (Tax/Config).
Delegation: The "Not My Job" principle (Order delegating to State).
